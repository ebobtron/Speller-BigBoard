Download the test program

testst5 is a modified version of the program that runs speller on each text file in
"/home/cs50/pset5/text" for this board. The source code is included in the zip file.

Usage is straightforward.

Unzip 'testst5' into your pset5 folder.
From a terminal window type

'cd Dropbox/pset5' then
'chmod 711 testst5' to set the permissions then use as below.

  ./testst5 ./speller [v]

  ./testst5 ./speller
run speller with all the .text files in ~cs50/pset5/texts.   Display average total times.

  ./testst5 ./speller v
run speller showing times for each text file with the average at the bottom

times are listed as below
total-time,  load-time,  check-time,  size-time,  unload-time.
